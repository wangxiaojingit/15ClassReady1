1、 复习

网页的基本构成

  • html :是超文本标记语言，因为它不仅可以表示普通的文本，还可以表示音频、视频、图片，它是一个网页的骨架。（骨架）

  • css：层叠样式表，它主要用来美化我们的网页，让其变得更美丽。（衣服）

  • js：javascript 简写，它呢主要用来表示一些行为或者动态，主用和后台交互数据，写复杂的业务逻辑，以及页面的一些事件。（行为和动态）

html的基本构成



<!DOCTYPE html>  //文档声明

<html>       

   <head>       //头部

      <meta charset="utf-8">  //编码格式，utf-8，全国通用，防止乱码

      <meta name="keywords" content="关键词，关键词"> //给搜索引擎看的  

      <meta name="description" content="描述的内容"> //给搜索引擎看的 

      <title>一个网页的标题</title>

   </head>

   <body>

      可视内容

   </body>

</html>

2、 第二天知识点

css引入样式的方式：

1、行内式，给谁添加样式，就写到谁的身上，要写到开始标签里（注意不要写到结束标签里），也要注意给标签名字有个空格隔开，例如：



  <h1 style="color:royalblue;">凤凰汽车</h1>

2、内嵌式，在head头部里面，写入一个style标签，在style标签里面是我们写样式的地方，给谁添加样式，就先选中谁，然后跟随一个{}，在{}里面写的就是咱们的css属性和属性值，例如：

<head>

    <meta charset="UTF-8">

    <title>Title</title>

    <style type="text/css">

        p{

            color:darkgreen;

        }

    </style>

</head>

如果页面中出现多个style，内嵌样式，我们的代码是从上而下执行的，如果一样的的情况下，后面的代码会覆盖前面的。

<style>

    /*内嵌式*/

    p{color:red;font-size:24px;background-color:darkcyan}

</style>

<style>

    p{color:green;}

</style>

3、外链式：新建一个css样式表，然后通过link标签，把css样式表链接到咱们的那个html文件，进行关联，这样咱们在css样式表中写的样式就会起作用了，例如：

   <head>

    <meta charset="UTF-8">

    <title>Title</title>

    <!--外链式-->

    <link rel="stylesheet" href="index.css">

</head>

4、导入式：

这种方式，咱们在工作中不用，它是一个命令语句

 <style type="text/css">

        @import "import.css";

    </style>

面试题： b、strong都是粗体，i、em都是斜体，他们有啥区别？
1、在语气上面，strong、em更具有语气话，也就是你们所说的语气更重一些，强调性更强些
2、更利于搜索引擎搜索优化

三大列表

1、有序列表：ol

 <ol>

      <li>周一</li>

      <li>周二</li>

      <li>周三</li>

      <li>周四</li>

  </ol>

2、无序列表: ul

<ul>

    <li>新闻一</li>

    <li>新闻二</li>

    <li>新闻三</li>

    <li>新闻四</li>

</ul>

3、自定义列表：dl

dt 代表的是小标题（列表中的每一项），dd代表的是对每个标题的描述，每个具体的小项

<h3>一日三餐</h3>

  <dl>

      <dt>早餐</dt>

      <dd>包子</dd>

      <dd>油条</dd>

      <dd>豆浆</dd>

      <dt>午餐</dt>

      <dd>麻辣烫</dd>

      <dd>鸡蛋面</dd>

      <dt>晚餐</dt>

      <dd>牛奶</dd>

  </dl>

总结：这三个列表的结构是固定的，ul(ol)面只能紧挨着的是li，li里面还可以放其他标签，dl里面只能是 dt、dd，是固定结构，但是dd里面还可以放其它标签，比如span

珠峰故事案例

   <!DOCTYPE html>

<html lang="en">

<head>

    <meta charset="UTF-8">

    <title>珠峰故事</title>

    <style type="text/css">

        /*

          * 代表通配符，意思是说选中所有的元素

          清除掉默认的padding margin值

        */

        *{

            padding:0;

            margin:0;

        }

        .main{

            width:830px;

            height:507px;

            background:#fff;

            margin:0 auto;/* 让块级元素水平居中 */

        }

        ul,li,ol{

            list-style: none; /*清除掉列表的默认样式 */

        }

        a{

            text-decoration:none;/*清除a标签下面的下划线*/

            font-size:14px;  /*字体大小*/

        }

        .story{

            font-size:18px;

        }

        ol span{

            background-color:#fd8c84;/*背景颜色*/

            font-size:12px;

            width:15px; /*宽*/

            height:15px;/*高*/

            display: inline-block;

            text-align: center;

        }

    </style>

</head>

<body>
