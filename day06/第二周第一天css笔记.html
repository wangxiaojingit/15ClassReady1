第二周第一天css笔记

把块级元素转换成行内块，让他们在一行显示，出现的一些问题。

  • 有一个间隙

  • 基线对齐的问题

解决间隙的问题

  • 消除回车换行，让代码在一行显示

  • 给它的父级加font-size：0；然后再给每个具体的元素设置文字大小。

       *{margin:0;padding:0}

        .main{

           font-size: 0;

        }

        .box1,.box2{

            width:100px;

            height:100px;

            background:green;

            display: inline-block;

            font-size:16px;

        }

   <div class="main">

       <div class="box1"></div>

       <div class="box2"></div>

   </div>

解决基线对齐

       *{margin:0;padding:0}

        .main{

           font-size: 0;

        }

        .box1,.box2{

            width:100px;

            height:100px;

            background:green;

            display: inline-block;

            font-size:16px;

        }



   <div class="main">

       <div class="box1"></div>

       <div class="box2">111112222</div>

   </div>

上面的代码，会出现盒子掉下的情况，这是因为行内元素或者行内块级元素它们默认的对齐方式是基线对齐，我们在上面的代码中加入：vertical-align: top; 就可以解决上面的问题，vertical-align 还有很多值，但是都不多常用。

浮动

浮动的应用场景： 如果想让块级元素在一行显示，我们就可以用浮动。
浮动常用的几个值：

  • float:left; 左浮动

  • float:right; 右浮动

  • float:none;默认值

浮动的特点：

  • 浮动的元素脱离文档流

  • 浮动元素的左右浮动，是参照父级的。

  • 浮动元素如果不设置宽度的时候，它不再继承父级的宽度，而是根据自身内容的大小。（在普通的文档流中，宽度是可以继承的，但是在浮动中，特例）

  • 造成父级的“盒子塌陷”，（父级没有了高度，因为子盒子脱离了文档流）

    • 面试题：解决浮动造成的盒子塌陷问题

    • 给父盒子增加高度

    • 给父级增加： overflow: hidden;

    • 在父级的盒子末尾添加clear：both，就可以解决。clear：left，清除左浮动，clear：right：清除右浮动，clear：both，清除左右浮动。

    • 用clearfix, 只要子盒子有浮动，就立马给它的父盒子添加类名：clearfix.(本质和第三个方法是一样的，只是为了我们在做项目时更加的方便，不用手动在浮动盒子的父级末尾添加div)

    • 浮动的本质，可以实现图文混编，如果有图片，有文字的时候，我们给图片设置浮动，然后文字就会自动的挨着图片进行环绕排列。

    注意：浮动，除了图文混编，咱们在做项目的时候，一般，如果给一个元素设置了浮动，那么它的兄弟元素通常都要设置，”就是说浮动了，就全浮动“，如果不这样做的话，会有很多问题。

            .clearfix:after{

                 display: block;

                 content: "";

                 visibility: hidden;

                 height: 0;

                 font-size:0;

                 clear: both;

             }

             .clearfix{

                 *zoom:1

             }

    

    

      <div class="main clearfix">

          <div class="box1">11111</div>

          <div class="box4">11111</div>

          <!--<div class="box3"></div>-->

      </div>

    overflow常用的

    • overflow:hidden; 超出隐藏

    • overflow:auto;如果内容超出盒子的大小，会自动出现滚动条

    • overflow：scroll；不管内容是不是超出盒子的大小，都会出现滚动条。

          .box{

                width:200px;

                height:200px;

                border:1px solid green;

                overflow:hidden;

                /*overflow:auto;*/

                /*overflow-x:auto;*/

                /*overflow-y:auto;*/

                /*overflow:scroll;*/

            }

    background

    • background-color; 背景颜色

    • background-image:url(“”) ; 背景图片

    • background-position; 背景图片的位置,让图片在盒子居中：center、center

      • 可以是具体的像素

      • 百分比

      • left、right、top、bottom、center

    • background-attachment:fixed/scroll;

      • fixed: 背景图片相对于窗口是固定的

      • scroll：背景图片相对元素是固定的

    • background-repeat: 控制背景图片是不是平铺，或者平铺的方向

      • no-repeat

      • repeat (默认)

      • repeat-x: 沿着x轴平铺

      • repeat-y:沿着y轴平铺

    • background-size: 设置背景图片的尺寸

      • 可以设置具体的值，一个代表x轴，即宽度，另一个代表y轴，即高度

      • 还可以设置百分比

      • cover：图片放大到铺满整个容器，有可能超出的部分不能完全显示在盒子里。

      • contain：是以合适的方式去放大图片，只要它的宽度或者高度达到容器的大小了，就停止了。

     .box{

                border:1px solid red;

                width:400px;

                height:300px;

                background: url("head.gif") center center no-repeat;

                /*background-size:300px 300px;*/

                /*background-size:100% 100%;*/

                background-size:cover;

                /*background-size:contain;*/

                background:red url("head.gif") center center fixed no-repeat;

            }

     background:red url("head.gif") center center fixed no-repeat;

    相对路径和绝对路径

    • 绝对路径：指的不管是外部还是内部都能访问到例如：带域名的路径，或者是电脑磁盘中的完整路径。

    • 相对路径，指的是以一个文件或者文件夹作为参考点，去寻找其它文件，然后写的路径. "../" 代表上一级。 如果是同级目录，直接写名字, 如果是下一级用“/”
