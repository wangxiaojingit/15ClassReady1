第四天css笔记

选择器

1、标签选择器

权重:1
标签选择器就是，通过标签名字，把这一类的标签都选中

  <p>p段落</p>

  <p>p段落</p>

  <p>p段落</p>

  <h2>h2</h2>

  <a href="#">百度</a>

2 通配符选择器

权重：1
用“*”来表示，代表选中所有的元素

 /*标签选择器*/

 p{color:green}

3、类选择器

权重:10
起名字：class

  • 想要选中谁，就给谁身上通过class给它起一个名字
    使用：“.” 加上名字
    通过小圆点，加上之前起的名字就可以了

 .p1{color:green}



 <p class="p1">p段落</p>

4、属性选择器

权重：10
同过给标签增加属性名字，或者属性名字等于属性值的方式，我们通过中括号来选取具有这类属性的元素或者属性名=属性值的一类元素。
属性名=属性值 ：类似 src、title 像这种的放到标签里面的就是一个元素的属性

<img src="" title="风景图片"/>

属性选择器的两种用法：1、直接属性名，2、属性名=属性值

       [title]{

            background:green;

        } 

        [title="aa"]{

            background:gold;

        }

5、id选择器

权重：100
选择谁，就给谁身上通过id起个名字，在使用的时候，我们通过#加名字的方式进行选中
例如

     <style>

       #title1{

           background:green;

       }



    </style>



<h2 >2级</h2>

6、子选择器

权重：所有选择器之和
通俗的说，就是儿子选择器，如果想要选中着父亲的子元素，我们可以通过”>” 把两者连接起来，这样就选中了符合条件的子元素
语法： ul>li

 .box>p{

            background:gold

 }

7、后代选择器

权重：所有选择器之和
通俗的说，所有的子子孙孙…… 都叫后代，我们通过空格把他们连接起来。
语法： div p{}

   .box span{

            color:green;

   }

  <div class="box">

      <p>段落</p>

      <h1>h1标题

          <span>span</span>

          <span>span</span>

          <span>span</span>

      </h1>

  </div>

8、分组选择器

权重：所有选择器之和
可以把具有相同样式的元素，通过逗号“，”连接起来，从而达到共同的样式效果，减少冗余的代码

    <style type="text/css">

        p,div,h1{color:red;background:green;}



    </style>



  <p>我是p1</p>

  <div>我是div</div>

  <h1>我是h1</h1>

9、相邻兄弟选择器

权重：所有选择器之和
选中与它相邻的那个兄弟元素，连接符“+”

      h1+h2{    // 有顺序，h1在前，h2在后

          color:green;

      }

<h1 id="title1">1级</h1>

<h1 >1级</h1>

<h2 >2级</h2>

<h3>3级</h3>

<h4>4级</h4>

<h5>5级</h5>

<h6>6级</h6>

10、通用兄弟选择器

权重：所有选择器之和
选中所有兄弟中，符合条件的兄弟元素，连接符“~”

       div h1~h3{

           color:red;

       }

    <h1 id="title1">1级</h1>

    <h1 >1级1111</h1>

    <h2 >2级</h2>

    <h2 >2级</h2>

    <h2 >2级</h2>

    <h3>3级</h3>

    <h3>3级</h3>

11、 伪类选择器

a 链接的四种状态

<style>

        a:link{          // a 标签默认的文字颜色

            color:red;

        }

        a:visited{     // a 标签访问过后的颜色

            color:green;

        }

        a:hover{      // 鼠标滑过之后的颜色

            color:pink;

        }

        a:active{    // 鼠标点击的文字颜色

            color:blue;

        }

    </style>
书写的时候是有顺序的，遵循“爱恨原则”，上面的只要了解即可，但是要记住hover

nth-child(n) n 可以使具体的数字，代表选中几个

       div p:nth-child(1){

            background:red;

        }

:nth-child(2n){} 代表的是偶数，也可以:nth-child(even){}
:nth-child(2n+1){}代表的是奇数，也可以:nth-child(odd){}
备注：2n 中的2不是死值，可以随意写
:first-child 选中第一个，:last-child 选中最后一个

        p:first-child{  

            color:gold;

        }

        p:last-child{

            background: green;

        }

选中 div 下面的类型是p的，然后顺序是p类型中第一个的元素

  div p:nth-of-type(1){

            background:green;

  }

table 表格

完整的table表格包括thead（头部）、tbody（身体）、tfoot（尾部），每一行用tr来表示，tr里面的每个单元格用td来表示，tr里面只能放td(th)，如果是标题，我们用th代替td; td里面可以放其它的比如：span

完整的表格

   <table border="1">

       <caption>成绩表</caption>

       <!--身体-->

       <tbody>

          <tr>

              <td>100</td>

              <td>100</td>

              <td>100</td>

          </tr>

          <tr>

              <td>100</td>

              <td>100</td>

              <td>100</td>

          </tr>

          <tr>

              <td>100</td>

              <td>100</td>

              <td>100</td>

          </tr>

       </tbody>

       <!--尾部-->

       <tfoot>

       <tr>

           <td>111</td>

           <td>111</td>

           <td>111</td>

       </tr>

       </tfoot>

       <!--头部-->

       <thead>

       <tr>

           <th>语法</th>

           <th>数学</th>

           <th>外语</th>

       </tr>

       </thead>



   </table>

简单形式

<table border="1" cellpadding="20" cellspacing="20">

      <caption>成绩表</caption>

      <tr>

          <th>语文</th>

          <th>数学</th>

          <th>外语</th>

      </tr>

      <tr>

          <td>100</td>

          <td>100</td>

          <td>100</td>

      </tr>

      <tr>

          <td>100</td>

          <td>100</td>

          <td>100</td>

      </tr>



      <tr>

          <td>100</td>

          <td>100</td>

          <td>100</td>

      </tr>



  </table>

border=”1” ,表格的边框线为1px
cellpadding=”20” 单元格的内填充
cellspacing=”20” 单元格与单元格的间距
在样式中加入：table-layout：fixed，平均分布；（前提必须有宽度）

 <style>

        table{

            margin:0 auto;

            /*边框合并*/

            border-collapse:collapse;

            width:500px;

            height:200px;

            /*平均分布*/

            table-layout: fixed;

            text-align:center;

        }

    </style>

面试题：完整写法的table和简写table的区别？

  • 更加有语义化，更利于搜索引擎搜索

  • 即使我们手动的更换thead、tbody、tfoot的位置，但是在浏览器中，他们的位置依然正常吧，所以即使我们的代码顺序没按照正常的写，也是可以的。但是我们通常不这样做，这样的话，浏览器就又麻烦了一次。
