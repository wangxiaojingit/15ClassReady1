盒模型

盒模型就是写网页时候的一种思维模式，每个元素都可以看成一个盒子，为了方便大家理解，我们可以把它当成一个“手机盒子”，里面的手机就是我们所说的“content”部分，手机盒子中的白色填充泡沫，就是padding，手机盒子的厚度，就是我们所说的“border”, 盒子与盒子之间的距离，就是我们所说的“margin”。
一个盒子模型包括：content（自身大小）、padding、border、margin
总盒子的大小height=content的高度+2倍的padding+2倍的border

<style>



        *{padding:0;margin:0;}

        div{

            width:200px;

            height:200px;

            background:green;

            padding:20px;

            border:20px solid gold;

            margin:30px;



        }

    </style>

这个盒子默认情况下，总盒子的大小height=content（代码中的height）+2*20+2*20=280；
如果在上面的代码变成的下面的

       *{padding:0;margin:0;}



      div{

            width:200px;

            height:200px;

            background:green;

            padding:20px;

            border:20px solid gold;

            margin:30px;

            box-sizing: border-box;



        }

我们发现如果在代码中加入box-sizing：border-box,我们上面的代码中的width 就是整个盒子的总宽度，盒子模型中的content部分就可以自己计算。
盒子模型的两种类型：

  • 标准盒模型：box-sizing：border-content; (默认)

  • ie盒模型 ： box-sizing: border-box;

margin

margin-top:10px; 上
margin-left：20px; 左
margin-right：30px;右
margin-bottom:40px; 下

border 和padding 于此类似，也有四个值，四个方向。

      .box1{

            /*四个值得情况：上、右、下、左*/

            /*margin:10px 30px 50px 100px;*/

            /*三个值得情况：上、左/右 下*/

            /*margin:10px 50px 100px;*/

            /*两个值的情况:上下、左右*/

            margin:10px 100px;

        }

padding

在content部分写的文字内容，是不会到padding部分的，只会在content部分，但是背景颜色和背景图片是整个大盒子就可以，所以padding经常用来做内边距。

面试题： margin 值穿透问题（只有上边距）

如果一个父盒子里面写了一个小盒子，然后如果父盒子没有border-top 也没有padding-top， 当我们给子盒子设置margin-top的时候，父盒子会跟着子盒子一起向下移动。
解决的办法：

  • 父盒子设置 overflow：hidden；

  • 不给子盒子设置margin-top，只用给父盒子设置padding-top，咱们可以同样达到效果

  • 给父盒子设置一个border-top: 1px solid transparent; 设置一个上边框，就可以解决，如果不想要颜色，可以设置成一个透明色，transparent

 <style>



        *{padding:0;margin:0;}

        .box1{

            /*padding-top:50px;*/

            box-sizing: border-box;

            width:200px;

            height:200px;

            background:pink;

            /*border-top:1px solid  red;*/

            border-top:1px solid  transparent;

            /*overflow: hidden;*/

        }



        .box2{

            width:50px;

            height:50px;

            margin-top:50px;

            /*padding:20px;*/

            background:green;

            /*margin-top:50px;*/

        }

    </style>



      <div class="box1">

          <div class="box2">2</div>

      </div>

margin 值合并的问题

在垂直方向上，margin’值合并问题，如果两个同级的块级元素，上边的一个设置margin-bottom：xxx； 下面的一个盒子设置margin-top：xxx, 两个盒子之间的距离，并不是他们的margin 之和，而是取最大的那个值。

所以我们在写代码的时候，注意一下，所以在写项目的时候，我们尽量在垂直方向上写一致就可以了。
