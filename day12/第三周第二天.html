了解移动端

产品形态：

  • pc和移动端两个域名的，做两套设计的，例如：京东，网易，淘宝

  • pc端和移动端公有一个域名的，例如：

    • 猎豹：https://www.liebao.cn/

    • 华为：https://msale.vmall.com/hwmate.html?cid=10600

    • 苹果：https://www.apple.com/cn/?afid=p238%7C11DPVC7K_mtid_18707vxu38484&cid=aos-cn-kwsg-brand

    什么时候公用一套，什么时候用一套？

    对于复杂的网站，我们通常用两套，对于结构简单，内容少的，我们通常用一套。

pc 和移动端的区别

效果或者事件

  • pc 有与鼠标相关的效果，如：鼠标滑动，点击等效果,移动端没有

  • 移动端有手指相关效果例如：触屏，滑屏等，pc没有

浏览器上

  • pc 端要考虑兼容问题，ie8（ie7）-ie11、谷歌、火狐

  • 移动端要解决手机适配问题，比如：机型不同，会出现不同的效果，还有手机屏幕不一样。(webkit内核，不用考虑ie兼容性问题)

  • 线性渐变

  • background:linear-gradient([方向|角度]？<点>,<点>);

第一个值，代表方向（left、top、right、bottom），或者角度（例如：45deg），线性渐变是成一条直线的，所以当我们写backgroundlinear-gradient(left,red,green)的时候，就是从左边开始，到右边，颜色从red到green过渡。

当我们写backgroundlinear-gradient(top,red,green)的时候，就是从上边开始，到下边，颜色从red到green过渡。

当我们写角度的时候，需要记住一个图，

 background:-webkit-linear-gradient(top,red 10%,green 80%);

10%之前，是red，80%之后是green，10% 到80%之间是red到green的过渡色



 background:-webkit-linear-gradient(top,red 10%,green 10%);

上面的代码，没有过渡色，10%之前是红色，10%之后的都是绿色

如果不想设置过渡色,我们只用设置一个起始位置就可以

```
background:-webkit-linear-gradient(top,red 10%,red 30%,green 30%,green 50%,blue 50%);

```

除了百分比，我们还可以设置具体的值

 background:-webkit-linear-gradient(top,red 10px,red 30px,green 30px,green 50px,blue 50px,blue 80px,yellow 80px);

线性渐变还可以当背景图片使用：

 background:-webkit-repeating-linear-gradient(top,red 10px,red 30px,green 30px,green 50px,blue 50px,blue 80px,yellow 80px,yellow 100px);

径向渐变

radial-gradient([shape|size][at ]?,|at ,]?[,]+])

background:radial-gradient(circle,red,green);

background:radial-gradient(ellipse,red,green);

第一个值定义的是形状，cirle 定义的是圆，ellipse定义的是椭圆

 background:radial-gradient(200px 100px,red,green);

当为具体的px的时候，200px 的位置为水平的半径，100px的位置为垂直半径，中间用空格分割

background:radial-gradient(at right top,red,green);

at right top 控制中心点的位置，右上角，可以随意设置组合，top，right，left，bottom

除了上面的位置，还可以设置具体的像素值。

 background:radial-gradient(at 50px 150px,red,green);

size:渐变的大小，即渐变到哪里停止，有如下关键词：closest-side:最近边，
farthest-side:最远边；closest-corner:最近角，farthest-corner:最远角(默认值)

 background:radial-gradient(farthest-side at 50px 150px,red,green);





background:radial-gradient(circle,red 10px,green 20px);

跟线性一样，在10px之前是red，20px之后是绿色，10px到20px之间是过渡

视口

在没加viewport 的时候，在移动端默认的html页面的宽度是980px;和我们的
手机的设备的宽度并不一致，这样的话，我们的页面要想在手机设备中显示完全，就会被缩放，但是这样的效果并不好，所以移动端我们需要加上viewport

viewport 快捷键：meta:vp

在做移动端的时候，我们必须加上下面的代码

<meta name="viewport"

          content="width=device-width, user-scalable=no, initial-scale=1.0, maximum-scale=1.0, minimum-scale=1.0">
名称 具体含义
width=device-width 页面的宽度等于设备的宽度
user-scalable=no 禁止缩放
initial-scale=1.0 初始缩放值为1.0
maximum-scale=1.0 最大缩放值为1.0
minimum-scale=1.0 最小的缩放值为1.0

### 媒体查询

```
@media screen and (min-width:375px){
.box{
width:100px;
height:100px;
background:green
}
}

```

and 符必须用空格隔开

media:媒体、媒介
媒体的类型：

  • screen :屏幕

  • print : 打印机

  • all ： 所有的设备

连接符：

  • and

  • only

  • not

    {} 里面存放的是css样式

    • 面试题：各大浏览器内核

    1、IE浏览器内核：Trident内核，也是俗称的IE内核；

    2、Chrome浏览器内核：统称为Chromium内核或Chrome内核，以前是Webkit内核，现在是Blink内核；

    3、Firefox浏览器内核：Gecko内核，俗称Firefox内核；

    4、Safari浏览器内核：Webkit内核；

    5、Opera浏览器内核：最初是自己的Presto内核，后来加入谷歌大军，从Webkit又到了Blink内核；

    6、360浏览器、猎豹浏览器内核：IE+Chrome双内核；

    7、搜狗、遨游、QQ浏览器内核：Trident（兼容模式）+Webkit（高速模式）；

    8、百度浏览器、世界之窗内核：IE内核；

    9、2345浏览器内核：好像以前是IE内核，现在也是IE+Chrome双内核了；

    10、UC浏览器内核：这个众口不一，UC说是他们自己研发的U3内核，但好像还是基于Webkit和Trident，还有说是基于火狐内核。。
